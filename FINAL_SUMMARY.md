## 🚀 Deploy in 3 Simple Steps

### Step 1: Get the Files (2 minutes)
```bash
# Option A: Extract the compressed archive
tar -xzf cap-springfield-inventory.tar.gz
cd cap-springfield-inventory

# Option B: Copy individual files maintaining folder structure
mkdir cap-springfield-inventory
cd cap-springfield-inventory
# Copy all files here with folders
```

### Step 2: Push to GitHub (3 minutes)
```bash
git init
git add .
git commit -m "Springfield CAP Inventory v1.0"
git remote add origin https://github.com/shawnwolfe25/cap-springfield-inventory.git
git push -u origin main
```

### Step 3: Deploy to Vercel (2 minutes)
1. Visit https://vercel.com
2. Click "Add New Project"
3. Select your GitHub repo
4. Click "Deploy"
5. Done! Your app is live

**Your URL:** https://cap-springfield-inventory.vercel.app/
