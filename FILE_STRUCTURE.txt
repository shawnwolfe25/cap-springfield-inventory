QUICK START COMMANDS
================================================================================

1. CLONE/COPY FILES TO YOUR MACHINE
   mkdir cap-springfield-inventory
   cd cap-springfield-inventory
   # Copy all files here

2. INITIALIZE GIT
   git init
   git add .
   git commit -m "Springfield CAP Inventory v1.0"

3. ADD GITHUB REMOTE
   git remote add origin https://github.com/shawnwolfe25/cap-springfield-inventory.git

4. PUSH TO GITHUB
   git push -u origin main

5. DEPLOY TO VERCEL
   • Visit vercel.com
   • Click "Add New Project"
   • Select the GitHub repository
   • Click "Deploy"

6. YOUR APP IS LIVE AT
   https://cap-springfield-inventory.vercel.app/

DEPLOYMENT OPTIONS
================================================================================

Option 1: BASIC (Local Storage Only)
  ✓ Deploy to Vercel
  ✓ No configuration needed
  ✓ Each user has separate data
  ✓ Perfect for single-user use
  Time to deploy: 5 minutes

Option 2: CLOUD (Vercel KV)
  ✓ Deploy to Vercel
  ✓ Add Vercel KV integration
  ✓ Set environment variables
  ✓ All users see same data
  Time to deploy: 10 minutes

Option 3: FULL (Cloud + Email)
  ✓ Deploy to Vercel
  ✓ Add Vercel KV
  ✓ Configure EmailJS
  ✓ Set all environment variables
  ✓ Automatic low-stock alerts
  Time to deploy: 15 minutes
