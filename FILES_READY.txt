Step 1: Extract Files (2 min)
  tar -xzf cap-springfield-inventory.tar.gz

Step 2: Push to GitHub (3 min)

  git add .

  git push -u origin main

Step 3: Deploy to Vercel (5 min)
  1. Go to vercel.com
  2. Click "Add New Project"
  3. Select your GitHub repo
  4. Click "Deploy"
