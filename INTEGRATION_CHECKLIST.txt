STEP 1: Add Importer Files (2 minutes)
───────────────────────────────────────
  □ Copy importer.js to your js/ folder
  □ Copy springfield_inventory_data.json to project root
  □ Update index.html - add before </body>:
    <script src="js/importer.js"></script>

STEP 2: Add Import UI (3 minutes)
─────────────────────────────────
  □ Open IMPORT_CODE_FOR_UI.js
  □ Copy all the code (starts with "showImportDataModal")
  □ Open js/ui.js
  □ Find the closing brace of UIManager class (before last "}")
  □ Paste the code there
  □ Add import button to Settings in ui.js renderSettings()

STEP 3: Deploy & Import (5 minutes)
────────────────────────────────────
  □ Push to GitHub (git add, commit, push)
  □ Deploy to Vercel (standard process)
  □ Wait for deployment to complete
  □ Open your app at vercel.com deployment
  □ Navigate to Settings page
  □ Click "📥 Import Existing Data" button
  □ Click "Select File" and choose springfield_inventory_data.json
  □ Click "Preview" (should show 149 items)
  □ Click "Import"
  □ Wait for import to complete
  □ See success message
  □ Go to Inventory page
  □ See all 149 items loaded!
